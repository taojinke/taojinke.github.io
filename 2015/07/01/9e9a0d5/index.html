OS X 10.10.4发布：提高网络可靠性，移除Discoveryd -

By edwin

发表于 2015-07-01

文章目录

苹果今天发布了 OS X Yosemite 10.10.4，这次升级对系统底层进行了优化，修复了一些严重的 Bug 并改进了性能。值得注意的是，OS X 10.10.4 移除了 Discoveryd 进程，Discoveryd DNS 解析进程在 OS X Yosemite 中造成了很多网络问题。OS X 10.10.4 可以通过 Mac App Store 完成更新。

移除了 Discoveryd DNS 解析进程后， OS X 10.10.4 重新开始使用 mDNSresponder 进程。Discoveryd 进程造成了很多问题，比如从休眠中唤醒速度减慢，无法街机 DNS，网络重名，WiFi 断开连接，CPU 占用过高，电池消耗过快等。OS X 10.10.4 测试进行了2个月的时间，第一个测试版在4月中旬发布。相信在 OS X El Capitan 发布前，我们还会迎来最后一个 OS X Yosemite 升级补丁。
