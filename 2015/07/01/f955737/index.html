ICEfaces EE 1.8.2.GA_P08 发布 -

发表于 2015-07-01

ICEfaces EE 1.8.2.GA_P08 发布，这是一个生产认证的维护版本，更新内容如下：

Notable ICEfaces Changes In This Release * ICEfaces Component Suite  

  • [ ICE-10492 ]            - ice:inputRichText - IE11 Support
  • [ ICE-10708 ]            - ice:selectInputText arrow keys selection not working            (IE11)

  • [ ICE-10355 ]            - ice:inputFile - update commons-fileupload library to            v1.3.1 for security fix

    • Note that the commons-fileUpload.jar v1.3.1 requires JDK              1.5+, and the commons-io.jar.  If you are deploying              to a JDK 1.4 environment, you must substitute v1.2.2 of              the commons-fileUpload.jar (available from the Apache                Commons FileUpload site ).

  • [ ICE-9604 ]             - ice:inputFile fileEntryListener/rowSelectListener            validation fails

  • [ ICE-1755 ]             - ice:outputLink incorrectly encodes space as "+" in path
  • [ ICE-10553 ]            - ice:panelGroup revert Draggable position when Escape key            is pressed while dragging
  • [ ICE-10699 ]            - ice:panelGroup revert feature doesn't position elements            correctly when using dragGhost or ghosting
  • [ ICE-10305 ]            - ice:panelCollapsible - Expanded panel sets inline padding

  • Core Framework

    • [ ICE-9583 ]            - Stuck threads reported with WeakHashMap.get() on Facelets            classes
    • [ ICE-9627 ]            - Deadlock during session invalidation
    • [ ICE-9804 ]            - CLONE - Deadlock on Session invalidation/logout on            WebSphere
    • [ ICE-9836 ]            - Classpath.getAlternativeJarFile(…) causes Bad file            descriptor under WebSphere 8.5
    • [ ICE-9842 ]            - ClassCastException thrown when loading non-ICEfaces pages

  • Samples

    • [ ICE-10410 ]            - Component Showcase image preloading fails on Chrome for            missing images
    • [ ICE-10412 ]            - Component-showcase Google Maps missing controls
    • [ ICE-10622 ]            - IE11 specific issues with samples/component-showcase

Notable ICEpack Changes In This Release

  • [ IPCK-472 ]          - cc:inputFile: JS error occuring in Composite Showcase File          Upload Demo
  • [ IPCK-474 ]          - EE Composite Showcase - File Upload cannot handle files          containing special characters
  • [ IPCK-488 ]          - richDataGrid - Decrease button icon title references the          increase title

  • [ IPCK-510 ]          - NPE thrown when starting EPS

  • Platform Support

    • Certify ICEfaces EE 1.8.2 on Internet Explorer 11 (IE 11).

更多内容请查看 发行日志 。

该版本下载请点： ICEfaces 1.x download page .

ICEfaces 商业版提供额外的特性和面向可靠和可伸缩的企业应用发布。ICEfaces EE 是完全测试和认证的企业级解决方案。

ICEfaces是一个基于Ajax的JSF开发框架。ICEfaces原本是一个商业产品，现已基于Mozilla Public License协议开源了。它提供一整套完整的Java EE应用程序开发组件，能够帮助开发人员用纯Java(not JavaScript)快速开发瘦客户端胖互联网应用程序(Rich Internet Applications:RIA)。另外可通过在线 Demo 体验一下ICEfaces的强大功能。
