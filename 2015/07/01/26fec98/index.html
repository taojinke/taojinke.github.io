Display status of all Concurrent Managers - Paul's Notes

By edwin

发表于 2015-07-01

文章目录 
rem -----------------------------------------------------------------------
rem Filename:   appinfo.sql
rem Purpose:    Script to display status of all the Concurrent Managers
rem Author:     Anonymous
rem -----------------------------------------------------------------------

set head on

column OsId       format A10 justify left
column CpId       format 999999
column Opid       format 999
column Manager    format A30
column Status     format A20
column Started_At format A30

column Cpid       heading &apos;Concurrent|Process ID&apos;
column OsId       heading &apos;System|Process ID&apos;
column Opid       heading &apos;Oracle|Process ID&apos;
column Manager    heading &apos;Concurrent Manager Name&apos;
column Status     heading &apos;Status|of Concurrent|Manager&apos;
column Started_At heading &apos;Concurrent Manager|Started at&apos;

select distinct Concurrent_Process_Id CpId, PID Opid,
       Os_Process_ID Osid, Q.Concurrent_Queue_Name Manager,
       P.process_status_code Status,
       To_Char(P.Process_Start_Date, &apos;MM-DD-YYYY HH:MI:SSAM&apos;) Started_At
from   Fnd_Concurrent_Processes P, Fnd_Concurrent_Queues Q, FND_V$Process
where  Q.Application_Id = Queue_Application_ID
  and  Q.Concurrent_Queue_ID = P.Concurrent_Queue_ID
  and  Spid = Os_Process_ID
  and  Process_Status_Code not in (&apos;K&apos;,&apos;S&apos;)
order  by Concurrent_Process_ID, Os_Process_Id, Q.Concurrent_Queue_Name
/
