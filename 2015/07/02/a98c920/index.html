文章目录
  1. 1. 移动端开发
    1. 1.1. 聊聊移动端跨平台开发的各种技术这篇文章绝对能够让您脑洞大开，移动端跨平台开发技术的普及，涉及到了很多知识点，大家都进来学习一下吧！
    2. 1.2. 移动WEB调试利器——Rosin腾讯alloyteam出品的移动端调试工具，基于fiddler的扩展，能用我们熟悉的fiddler来输出控制台的内容，更强大的是还能捕捉到控制台的报错信息。
    3. 1.3. 玩转HTML5移动页面（动效篇）这篇文章介绍了使用CSS3，SVG等技术来制作丰富HTML5移动页面，不仅能看到别人页面那么酷炫，自己也能学着���一个了。
    4. 1.4. 玩转HTML5移动页面（优化篇）你是不是看了很多移动端页面优化的文章，但是都只是一个目录，没有干货，好吧，这里就有干货，除了目录还有如果实践来达到效果。按照页面制作流程来写的，很方便理解。
  2. 2. 奇舞推荐
    1. 2.1. Qnext前端交互沙龙去哪儿网前端交互沙龙第一期内容精彩呈现。内容涉及深入理解https、开发网页另一种选择——Canvas、前后端分离的实践和思考、Node.js上线与运维以及WebGL带你走进浏览器的3D世界。相信你们通过这五部分的分享，能学到很多前端方面的知识。
    2. 2.2. 团队使用Git和Git-Flow手记一个项目的开发，不仅仅依赖于代码的强大，代码的管理也尤为重要，本文非常细致的介绍了git在项目中的应用
    3. 2.3. 改进 ThinkJS 的异步编程方式ThinkJS 是奇舞团开源的一款 Node.js MVC 框架，该框架底层基于 Promise 来实现，很好的解决了 Node.js 里异步回调的问题。利用生成器函数和 Q 框架，可以方便地把 Promise 嵌套变成平级，让你的 ThinkJS 异步代码变得更清晰明了！
    4. 2.4. 用npm-run自动化任务你也许了解过Makefile，能够用来构建和管理工程。在这里文中介绍npm run命令相关的知识，用来自动构建和监控JavaScript、CSS。
  3. 3. 协议
    1. 3.1. 【篇一】Referrer Policy 介绍2014 年，W3C 的 Web 应用安全工作组（Web Application Security Working Group）发布了 Referrer Policy 草案，本文是对这个草案的介绍。有趣的是这个草案和 Chrome 当前的实现都有问题，欲知详情请点击~
    2. 3.2. 【篇二】Referrer 还是 Referer？Referrer 这个单词到底怎么拼，为什么有时候中间有两个 r，有时候只有一个？这是一篇科普小文，看完你就明白了~
  4. 4. 其它
    1. 4.1. 计算机之子Winter向大家推荐：移动Web手册本书为Peter-Paul Koch (PPK)新编写的《The Mobile Web Handbook》由360前端团队-奇舞团翻译小组翻译，中文名为：《移动Web手册》。
